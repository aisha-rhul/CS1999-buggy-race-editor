6-FREE Add custom features to the editor

I added the following additional features:

  1. Sliding buttons - These were added to the buggy-form in a replacement of the standard checkboxes to choose options fireproof, insulated, antibiotic and banging. These buttons look more appealing and give a good visual indication of the choice with clear RED and GREEN colours.

  2. Selecting a buggy - I decided to allow a user to select a buggy from a list rather than type in a buggy id as input when editing/deleting a buggy. This improved the usability of the software and gave a good visual indication of the available buggies adn their properties.

  3. Navigation Bars - Added a navigation bar on several pages to improve the user interface. It looks more in line with other websites in the internet.

  4. Change password - I thought it was absolutely necessary to include this feature as a user may want to change their password, especially after a password reset when a random password is received via email.

  5. User registration - This option was a must as new users will not be able to participate.

All of the above are tested to be working well.
