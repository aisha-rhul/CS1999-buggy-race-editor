0-GET Get the source code

I first installed the latest copy of Python (v3.8). To get the source code, I downloaded and installed git on my machine and then cloned a copy of the project from GitHub at: https://github.com/RHUL-CS-Projects/CS1999-buggy-race-editor. This created a local copy on my machine in a folder CS1999-buggy-race-editor.

0-RUN Get app running and view it in a browser

I then created a virtual environment venv and installed all the required components listed in requirements.txt using pip. Following this, I created the database file by running the given python script init_db.py . I set the environment variable for Flask as FLASK_ENV="development".

Running the app.py file started the Flask server. I then used a browser to view the site at http://127.0.0.1:500. All worked well.


0-CHANGE Make a change to a template and see it appear

I made a small change to the index.html by adding the words "Buggy Options". I then checked that the change took place in the browser to confirm. I made my first commit on 15 May 20 to GitHub. This change was reflected on GitHub at: https://github.com/aisha-rhul/CS1999-buggy-race-editor
